# Ohanna-Agent v1.0.0

Ohanna-Agent v1.0.0 constitue la première version prête pour un déploiement en production.

Ohanna-Agent observe l’infrastructure Ohanna, évalue ses capacités et produit des observations structurées destinées notamment à Ohanna-Vision.

## Fonctionnalités principales

* cycle de vie applicatif structuré ;
* configuration déclarative et validée ;
* modèle d’infrastructure composé de nœuds, services et endpoints ;
* moteur de capacités ;
* ordonnanceur de vérifications ;
* gestionnaire de plugins ;
* observation des capacités réseau ;
* plugin DNS ;
* génération et sérialisation des observations ;
* pipeline d’export ;
* publication événementielle ;
* connecteur vers Ohanna-Vision ;
* client HTTP de production ;
* point d’entrée CLI `ohanna-agent` ;
* configuration Linux standardisée ;
* service systemd ;
* scripts d’installation et de mise à jour ;
* journalisation et arrêt contrôlé ;
* packaging wheel et source distribution.

## Installation

```bash
python -m pip install ohanna_agent-1.0.0-py3-none-any.whl
```

Vérification :

```bash
ohanna-agent --version
ohanna-agent --help
```

## Déploiement Linux

Les ressources de déploiement sont incluses dans la distribution source :

* script d’installation ;
* script de mise à jour ;
* unité systemd ;
* exemples de configuration ;
* documentation de l’arborescence Linux.

Consulter la documentation d’installation et de déploiement avant toute mise en production.
